mysql 主键与外键

先建一个t1表，主键如果是UNSIGNED，那么外键关联的字段要也要是UNSIGNED，反正就是要一样。

外键名不能用id1，即： CONSTRAINT id1会报错， 不知道为什么！！！

想一次在mysqlworkbench上运行，要设置运行出错继续运行。因为，初次运行没有这两张表就会报错。在菜单Query下设置。
